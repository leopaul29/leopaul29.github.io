Design and development of a Web application for testing purpose based on an already existing solution. Users can create new tests such as ping, SQL request, web services call. The application will perform with recurrence the tests and notify stakeholders by message using OpenFire server, mail or phone call.

Objectif: Integrate customer part in the current solution

Features implemented:
* Bugs fixing in display 
* Create new type of test for phantomJS script testing
* Rework of message service using OpenFire server
* Add layer customer account/role 
* Implementation of a Thread Pool to launch multiple test in parallel
* Add Internationalisation

Technical environment: Java 1.6, Eclipse, JavaEE, Spring, Hibernate 3, SOAPUI, Bootstrap 2.3, HTML, CSS, Javascript, JQuery Linux, Server OpenFire, MySQL 5.5
